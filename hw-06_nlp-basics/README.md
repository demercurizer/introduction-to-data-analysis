# HW‑06 — Основы обработки текстов (NLP)

Краткое содержание: подготовка текстов, векторизация (TF‑IDF, n‑граммы), базовые модели для текстовой классификации.

- Состав: `main.ipynb`
- Темы: очистка текста, токенизация, TF‑IDF, n‑граммы, логистическая регрессия/наивный Байес
- Навыки: построение NLP‑пайплайна, интерпретация признаков, оценка качества

Как запустить: откройте и выполните `main.ipynb`
