# HW‑03 — Линейная и логистическая регрессии (части 1–2)

Краткое содержание: построение и оценка линейных и логистических моделей. Внимание к выбору признаков, регуляризации и метрикам качества. Есть две части (легкая и сложная).

- Состав: `part-1.ipynb` (легкая), `part-2.ipynb` (сложная), архивные версии — в `archive/`
- Темы: линейная регрессия, логистическая регрессия, регуляризация, метрики (RMSE/ROC‑AUC и др.), кросс‑валидация
- Навыки: постановка экспериментов, аккуратные сравнения, интерпретация коэффициентов

Как запустить:
1) Установите зависимости (см. корневой README)
2) Запустите интересующую часть, начиная с `part-1.ipynb`
