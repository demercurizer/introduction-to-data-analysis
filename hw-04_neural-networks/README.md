# HW‑04 — Нейронные сети (части 1–2)

Краткое содержание: построение и анализ базовых нейронных сетей для задач классификации. Визуализация границ решений, регуляризация, контроль переобучения. Доп. материалы — в `notes/` и `artifacts/`.

- Состав: `part-1.ipynb`, `part-2.ipynb`; в `archive/` — предыдущие версии
- Темы: полносвязные сети, функции активации, регуляризация, визуализация качества
- Навыки: воспроизводимые эксперименты, анализ обучаемости, ясные выводы по результатам

Как запустить:
1) Установите зависимости (см. корневой README)
2) Выполните `part-1.ipynb`, затем `part-2.ipynb`
