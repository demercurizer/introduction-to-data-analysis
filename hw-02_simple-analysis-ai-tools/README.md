# HW‑02 — Простой анализ данных и ИИ‑инструменты

Краткое содержание: решение задач простого анализа данных с использованием классических библиотек и ИИ‑инструментов для ускорения рутинных частей и проверок. В папке `notes/` лежит использованный промпт.

- Состав: `main.ipynb`, `notes/ai_prompt.txt`
- Темы: Pandas, визуализация, проверки гипотез (базовые), помощь AI в написании кода
- Навыки: продуктивное сочетание ИИ и ручного анализа, прозрачная документация

Как запустить:
1) Установите зависимости (см. корневой README)
2) Откройте и выполните `main.ipynb`
