# HW‑07 — Кластеризация и понижение размерности

Краткое содержание: исследование неконтролируемого обучения: кластеризация (например, K‑Means) и понижение размерности (например, PCA). Визуализация групп и анализ устойчивости.

- Состав: `part-1.ipynb`, `part-2.ipynb`, `part-3.ipynb`
- Темы: K‑Means, иерархическая кластеризация, PCA/TSNE, визуализации кластеров
- Навыки: постановка экспериментов без разметки, анализ структуры данных, интерпретация

Как запустить: выполните интересующую часть по порядку (1 → 3)
