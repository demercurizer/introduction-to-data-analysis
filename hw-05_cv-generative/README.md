# HW‑05 — Компьютерное зрение и генеративные модели

Краткое содержание: задачи из области CV и генеративных подходов. Чистый и воспроизводимый код, фокус на качестве и интерпретации результатов.

- Состав: `main.ipynb`
- Темы: базовые пайплайны CV, аугментации, представления признаков, введение в генеративные модели
- Навыки: аккуратная обработка данных, контроль экспериментов, визуализации качества

Как запустить: откройте и выполните `main.ipynb` (см. корневой README для зависимостей)
