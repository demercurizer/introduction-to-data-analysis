# HW‑01 — Простой анализ данных

Краткое содержание: базовая работа с табличными данными (загрузка, предобработка, агрегации), первичный разведочный анализ (EDA) и визуализации. Фокус на аккуратном пайплайне и интерпретабельности.

- Состав: `main.ipynb`
- Темы: Pandas, визуализация (Matplotlib/Seaborn), сводные таблицы, простые метрики
- Навыки: чистый код в ноутбуках, воспроизводимые эксперименты, структурированный EDA

Как запустить:
1) Активируйте окружение и установите зависимости из корня репозитория
2) Откройте `main.ipynb` в этой папке и выполните ячейки по порядку
